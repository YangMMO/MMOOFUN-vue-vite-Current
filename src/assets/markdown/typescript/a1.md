## 基本开发环境
node npm vscode cmd chrome

<a name="RAzm3"></a>
## 安装
npm install -g typescript

<a name="WcZRu"></a>
## 编译
tsc ***.ts

<a name="TqnHx"></a>
## tsconfig.json配置
```json
{
  "compilerOptions": {  //编译选项
    "target": "es2015",
    "module": "commonjs",
    "outDir": "lib"
  },
  "include": [  //源码位置
    "src/**/*"
  ]
}
```

<a name="pPhnM"></a>
## 通过package.js编译
```json
"scripts": {
  "start": "tsc --watch", //实时监听文件变化，自动编译
  "build": "tsc", //编译一次
  "test": "echo \"Error: no test specified\" && exit 1"
},
```
